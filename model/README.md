# Model Directory

Place trained model files here:

- `eye_cnn.h5`  — Keras/TensorFlow CNN (generated by `utils/cnn_classifier.py:train_model()`)
- `shape_predictor_68_face_landmarks.dat` — dlib landmarks (optional, download from dlib.net)

## Without model files

The system runs in **EAR-only mode** using Haar cascades.
This is fully functional — the CNN only adds a second confirmation layer.

## Training

```bash
# Prepare dataset:
#   data/eyes/open/   — JPG/PNG images of open eyes
#   data/eyes/closed/ — JPG/PNG images of closed eyes

python -c "from utils.cnn_classifier import train_model; train_model('data/eyes')"
```

## Recommended Datasets

- MRL Eye Dataset: http://mrl.cs.vsb.cz/eyedataset
- CEW Dataset (Closed Eyes in the Wild)
- Drowsiness Detection Dataset on Kaggle
